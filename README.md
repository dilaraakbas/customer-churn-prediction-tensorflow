# 📊 Customer Churn Prediction with TensorFlow

This project uses an **Artificial Neural Network (ANN)** built with **TensorFlow** to predict whether a customer will leave the company (Churn).

## 🚀 Key Features
- **Data Pipeline:** Synthetic customer data generation with NumPy.
- **Deep Learning Architecture:** Multi-layer Perceptron (MLP) with ReLU and Sigmoid activations.
- **Evaluation:** Monitoring for overfitting with Train/Val/Test splits.
- **Business Insight:** Includes a strategic summary for management.

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- NumPy & Scikit-Learn
- Matplotlib (for visualization)

## 📈 Results
The model achieved an accuracy of approximately **80-85%** on the test set.
